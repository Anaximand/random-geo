# random-geo
Generating random dungeons using geomorphs

## GitHub Pages

This app is configured for the `Anaximand/random-geo` project Pages URL:

```text
https://anaximand.github.io/random-geo/
```

Deployments run from `.github/workflows/deploy.yml` on pushes to `main`. In the GitHub repository settings, set **Pages > Build and deployment > Source** to **GitHub Actions**.
